# 鄂尔多斯市银某公司诉杭锦旗自然资源局行政许可案
## 实施行政许可不能违法增设条件或者增加限制
### 关键字
行政 行政许可 建设工程规划许可 法定条件 法定事由 增设条件
### 基本案情
　　鄂尔多斯市银某公司于2019年8月竞买取得公开挂牌拍卖的（编号为2019-16，用途为加油加气的商业用地）宗地一块，2021年11月19日取得不动产权登记证书。后鄂尔多斯市银某公司申请以“银某加油站加气站扩建项目”使用该土地，取得用地规划许可、用地审批及项目备案，并编制了建设工程设计方案等材料。2022年3月，鄂尔多斯市银某公司向杭锦旗自然资源局提交了《关于建设工程规划许可证一次性告知单》所要求的全部资料，申请办理建设工程规划许可证。杭锦旗自然资源局于2022年8月5日作出《“银某加油站加气站扩建项目”审议意见通知单》，载明：“银某加油加气站扩建项目”于2022年8月4日经杭锦旗城乡规划管理委员会2022年第3次会议审议，因项目存在合伙纠纷问题未能通过，故不予办理《建设工程规划许可证》。鄂尔多斯市银某公司提起行政诉讼，请求：撤销杭锦旗自然资源局于2022年8月5日作出的审议意见通知单，判令杭锦旗自然资源局为鄂尔多斯市银某公司办理建设工程规划许可证。
　　内蒙古自治区鄂尔多斯市康巴什区人民法院于2023年3月3日作出（2023）内0603行初1号行政判决书，判决：一、撤销被告杭锦旗自然资源局于2022年8月5日作出的《“银某加油站加气站扩建项目”审议意见通知单》；二、责令被告杭锦旗自然资源局在20日内对原告鄂尔多斯市银某公司办理建设工程规划许可证的申请重新作出处理。宣判后，双方当事人在法定期限内均未提出上诉，判决已发生法律效力。
### 裁判理由
　　法院生效裁判认为，根据《中华人民共和国行政许可法》第四条规定，设定和实施行政许可，应当依照法定的权限、范围、条件和程序。《中华人民共和国行政许可法》第三十一条第一款规定，申请人申请行政许可，应当如实向行政机关提交有关材料和反映真实情况，并对其申请材料的实质内容真实性负责。行政机关不得要求申请人提交与其申请行政许可事项无关的技术材料和其他材料。即行政机关应严格依法实施行政许可，不能增加法律法规规定之外的条件或限制。《中华人民共和国城乡规划法》第四十条第二款规定，申请办理建设工程规划许可证，应当提交使用土地的有关证明文件、建设工程设计方案等材料。需要建设单位编制修建性详细规划的建设项目，还应当提交修建性详细规划。对符合控制性详细规划和规划条件的，由城市、县人民政府城乡规划主管部门或者省、自治区、直辖市人民政府确定的镇人民政府核发建设工程规划许可证。本案中，鄂尔多斯市银某公司提供的办理建设工程规划许可证的材料齐全，存在合伙纠纷并不是不能办理规划许可的法定事由，杭锦旗自然资源局以存在纠纷为由，不予办理《建设工程规划许可证》缺乏法律依据。依照《中华人民共和国城乡规划法》及《中华人民共和国行政许可法》的上述规定，鄂尔多斯市银某公司的诉讼请求应予支持。
### 裁判要旨
　　《中华人民共和国行政许可法》第四条明确规定行政机关实施行政许可，应当依照法定的权限、范围、条件和程序。《中华人民共和国城乡规划法》亦明确办理建设工程规划许可需提供的材料。行政机关实施行政许可，应当对行政许可申请是否符合法定条件进行判断，不能在法定条件之外增设条件或者增加限制。行政相对人申请办理建设工程规划许可符合法定条件的，行政机关应当依法办理行政许可。

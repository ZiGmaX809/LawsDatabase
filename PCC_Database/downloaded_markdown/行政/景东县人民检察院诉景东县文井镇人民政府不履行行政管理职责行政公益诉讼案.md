# 景东县人民检察院诉景东县文井镇人民政府不履行行政管理职责行政公益诉讼案
## 经过双方协商，公益诉讼起诉人诉讼请求已全部实现的，依法准许撤回起诉
### 关键字
行政 行政公益诉讼 行政管理职责 传统村落保护 实现诉讼请求 撤回起诉
### 基本案情
　　景东县境内的清凉村梁家组传统村落始建于清代，南北宽约600米，东西宽约200米，内有祠堂、梁家大院、吊脚楼、古戏楼等，全组85%以上居民仍保留原有的建筑风格，属第一批列入中国传统村落名录的国家级传统村落。景东县人民检察院以梁家组古村落存在不同程度的自然损坏、消防设施缺失、村民消防安全意识淡薄等问题，文井镇人民政府作为责任主体未全面依法履行职责为由，于2021年6月16日发出检察建议，督促其完善消防、防灾避险等必要的安全设施，整治文化遗产周边公共环境。同年7月9日，景东县文井镇人民政府书面回复景东县人民检察院已落实整改。同年11月7日，景东县人民检察院联合景东县消防救援大队对清凉村梁家组传统村落进行检查时发现，该村落存在线路老化、消防器材不完备、村民消防安全意识淡薄等情形，仍然存在消防安全隐患。景东县人民检察院遂向跨行政区域集中管辖的景谷傣族彝族自治县人民法院提起行政公益诉讼。
　　法院在审理过程中，经与双方充分沟通、协调，督促被告景东县文井镇人民政府对清凉村梁家组传统村落存在消防安全隐患的问题进行了整改，消除安全隐患。被告景东县文井镇人民政府针对人民法院反馈的问题进行了全面整改。
　　因公益诉讼起诉人的诉讼请求已全部实现，景东县人民检察院决定撤回起诉。云南省景谷傣族彝族自治县人民法院于2022年5月31日作出（2022）云0824行初1号行政裁定：准许公益诉讼起诉人景东彝族自治县人民检察院撤回起诉。
### 裁判理由
　　法院生效裁判认为，被告景东县文井镇人民政府已履行法定职责，消除传统村落存在的消防安全隐患，有效维护了国家利益和社会公共利益。公益诉讼起诉人的诉讼请求已经实现，撤回对被告的起诉符合法律规定，予以准许。
### 裁判要旨
　　对于诉讼请求是否全部实现的认定，需结合公益诉讼请求的类型和目的予以综合判断。本案系行政机关未全面履行传统村落管理和保护职责引发的行政公益诉讼，诉讼请求是否实现应以行政机关依法全面适当履职，有效维护社会公共利益为审查判断标准。本案中，人民法院通过审查消防管理部门出具的安全评估报告，结合实地勘查情况，传统村落存在的消防安全隐患确已消除，故依法准许撤回起诉。人民法院通过积极延伸审判职能，加强与行政机关、检察机关的协调联动，共同促进传统村落的管理维护和历史文化的传承发展。

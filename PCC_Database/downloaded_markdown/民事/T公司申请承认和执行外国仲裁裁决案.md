# T公司申请承认和执行外国仲裁裁决案
## 申请承认和执行外国仲裁裁决案件中，对仲裁庭超裁情形的处理
### 关键字
民事 申请承认和执行外国仲裁裁决 承认与执行外国仲裁裁决公约 超裁 裁决内容是否可分
### 基本案情
　　    某航运公司与T公司2010年至2011年间签订了HPO40至HPO45共六份集装箱租赁协议，约定由某航运公司承租T公司的集装箱，每份协议中还约定合同履行过程中产生的争议均适用美国加利福尼亚州法律并在旧金山市进行仲裁。某建投公司仅在其中一份协议即HPO42号协议中以共同承租人的身份签字，并无条件、不可撤回地授权某航运公司代表其行使与本协议有关的权利，承诺对该协议相关的费用以及产生的损失与某航运公司承担连带责任；对另外五份协议，某建投公司均未签字或者以其他形式作出任何类似的承诺。后因某航运公司拖欠租金，T公司依据前述六份协议向美国国际争议解决中心（以下简称ICDR）提交了仲裁申请，请求某航运公司、某建投公司连带清偿拖欠的租金和解除合同的损失等共计6581万美元。
　　    在仲裁过程中，某航运公司与T公司签订了一份《违约处理协议函》，该协议函对六份协议项下集装箱租赁费用以及违约责任进行了合并处理与确认，但未重新约定仲裁条款，而是声明因相关协议履行产生的纠纷仍依据原协议约定进行仲裁，某航运公司代理人代表某建投公司参与了该协议函的签署，并特别注明其授权源于“HPO42协议”。
　　    2014年12月30日，ICDR依据《违约处理协议函》对涉案六份协议项下纠纷作出最终裁决。因该协议函并未载明每份集装箱租赁协议项下承租人所负担的债务金额，故ICDR依据该协议函所作出的裁决也未明确每份协议项下的债务金额，而是直接裁决某航运公司和某建投公司就案涉六份协议合计向T公司连带赔偿6581万美元。
　　    申请人T公司根据上述仲裁裁决，向海口海事法院请求依法承认和执行ICDR作出的案涉仲裁裁决，对某航运公司、某建投公司予以强制执行。
　　    被申请人某航运公司、某建投公司辩称，ICDR将不是协议当事人的被申请人某建投公司纳入仲裁范围进行仲裁，且将六份协议合并仲裁，裁决被申请人某建投公司需对全部六份协议承担连带赔偿责任，合并程序违法且超越仲裁权限，违反《承认及执行外国仲裁裁决公约》第五条第一款的规定。请求对案涉仲裁裁决不予承认和执行。
　　    海口海事法院于2019年12月30日作出（2015）琼海法他字第1号民事裁定：一、对美国国际争议解决中心（ICDR）作出的案涉仲裁裁决涉及某航运公司裁决的部分，予以承认和执行；二、驳回T公司的其他请求。
　　
### 裁判理由
　　   本案的争议焦点为：本案是否存在仲裁庭超裁情形，以及如果存在超裁情形，对其是否应予承认和执行。
　　     一、本案存在仲裁庭超裁情形
　　    被申请人某建投公司仅为案涉HPO42号协议的缔约方，并非其他五份协议的缔约方，故HPO42协议之外的其他五份协议中的仲裁条款对被申请人某建投公司不具有约束力。仲裁庭仅能依据HPO42号协议中的仲裁条款，对被申请人某建投公司在该份协议项下所应承担的责任作出裁决。案涉仲裁裁决将被申请人某建投公司的责任扩展至另外五份协议，要求被申请人依据六份协议一并向申请人T公司承担责任，实际是仲裁庭针对某建投公司将该公司在HPO42号协议约定的仲裁范围之外的事项进行了裁决，属于超范围仲裁的情形。
　　    二、应根据超裁部分与有权裁决部分是否可分，具体判断是否承认和执行
　　   本案系申请承认和执行外国仲裁裁决纠纷案件。中华人民共和国与裁决地所在国美国均为《承认及执行外国仲裁裁决公约》的缔约国，根据《中华人民共和国民事诉讼法》（2017年修正）第二百八十三条的规定，案涉仲裁裁决能否得到我国的承认及执行，应当适用《承认及执行外国仲裁裁决公约》的规定进行审查。《承认与执行外国仲裁裁决公约》第五条第一款第（三）项规定，如果存在“裁决所处理之争议非为交付仲裁之标的或不在其条款之列，或裁决载有关于交付仲裁范围以外事项之决定者”，该仲裁裁决不得承认和执行，但“交付仲裁事项之决定可与未交付仲裁之事项划分时，裁决中关于交付仲裁事项之决定部分得予承认及执行”。
　　    本案中，因《违约处理协议函》是对六份协议项下债权债务的合并处理，没有列明每份协议项下的债务金额，仲裁庭据此作出的仲裁裁决中也未能载明承租人在每份集装箱租赁协议项下负担的债务金额，被申请人某建投公司在HPO42号协议项下的债务金额亦无法确定，由此导致裁决结果中涉及被申请人某建投公司责任的超裁部分与有权裁决部分无法分割，故对于案涉仲裁裁决涉及被申请人某建投公司的裁决内容，应当全部不予承认和执行；但某航运公司系六份协议的当事人，应对六份协议项下的全部债务承担责任，该责任承担无需对每份协议项下的债务金额进行区分，故该仲裁裁决中涉及某航运公司的裁决内容，与涉及某建投公司的裁决内容具有可分性，不存在不予承认和执行的情形，故对涉及某航运公司的裁决内容应当予以承认和执行。
### 裁判要旨
　　在申请承认和执行外国仲裁裁决案件中，仲裁庭裁决涉及的主体和事项超出仲裁协议约定的仲裁范围，超裁部分与有权裁决部分不可分的，对该仲裁裁决应当全部不予承认和执行；超裁部分与有权裁决部分可分的，对超裁部分应当不予承认和执行，但是对有权裁决部分则应当予以承认和执行。
